Aplicación Web "Gestor Químico"

Objetivo: Plataforma Fullstack que permite gestionar una base de datos de compuestos químicos, incluyendo un módulo que calcula automáticamente el peso molecular de un compuesto al ingresar su fórmula.

Hito 1: Lógica (Back-End Core)
Calculadora Molecular Avanzada

Énfasis: Uso de POO y Algoritmo Stack para parsear fórmulas (ej. Ca(OH)2).

Tecnologías: Python | POO | Algoritmos

Código Fuente (GitHub) Video Tutorial (YouTube)
Hito 2: Arquitectura de Datos
Modelo Relacional MySQL (8 Tablas)

Énfasis: Diseño ERD con Relaciones 1:1, 1:N y N:M para autonomía de datos.

Tecnologías: MySQL | SQL | ERD

Código Fuente (GitHub) Video Tutorial (YouTube)
Hito 3: Integración Full Stack
Plataforma Web Django con Autenticación y Autonomía

Énfasis: Consolidación Back-End, ORM para CRUD y sistema de Autenticación/Roles (Admin/Colaborador).

Tecnologías: Django | ORM | HTML/CSS | Seguridad

Código Fuente (GitHub) Video Tutorial (YouTube)
Oportunidades de Mejora

El proyecto es funcional, pero una mejora clave para su escalabilidad sería refactorizar la lógica del backend a una API (Django REST Framework). Esto desacoplaría el frontend y permitiría, a futuro, integrar librerías de Ciencia de Datos (Pandas/NumPy) para análisis más complejos.
