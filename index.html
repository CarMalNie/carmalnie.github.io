Aplicación Web "Gestor Químico"

Objetivo: Plataforma Fullstack que permite a usuarios registrados gestionar una base de datos de compuestos químicos, incluyendo un módulo que calcula automáticamente el peso molecular de un compuesto al ingresar su fórmula (basado en la IUPAC).

Hito 1: Lógica (Back-End Core)
Calculadora Molecular Avanzada
  • Énfasis: Uso de POO y Algoritmo Stack para parsear fórmulas (ej. Ca(OH)2).
  • Habilidades: Python, POO, Algoritmos, Manejo de Errores.
Ver Código (GitHub) Ver Video-Tutorial (Hito 1)
Hito 2: Arquitectura de Datos
Modelo Relacional MySQL
  • Énfasis: Diseño de Esquema de Base de Datos (ERD) normalizado con Relaciones M:N (Muchos a Muchos).
  • Habilidades: MySQL, SQL Avanzado, Modelado ERD.
Ver Código (GitHub) Ver Video-Tutorial (Hito 2)
Hito 3: Integración Full Stack
Plataforma Web Django con Autenticación
  • Énfasis: Integración Django/MySQL usando el ORM para el CRUD y sistema de Autenticación.
  • Habilidades: Django (MVT), ORM, Seguridad.
Ver Código (GitHub) Ver Video-Tutorial (Hito 3)
Oportunidades de Mejora

El proyecto es funcional, pero una mejora clave para su escalabilidad sería refactorizar la lógica del backend a una API (Django REST Framework). Esto desacoplaría el frontend y permitiría, a futuro, integrar librerías de Ciencia de Datos (Pandas/NumPy) para análisis más complejos.
